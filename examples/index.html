Buttons

Normal buttons

Buttons look can be applied to all kind of elements by using the .button class.

A button
Div button
Span button
<a href="#" class="button">A button</a>

Small buttons

<button type="button" class="button button--small">A small button</button>

Button variations

<button type="button" class="button button--ghost">A ghost button</button>

Forms

Every form line is wrapped in a .form-group this is what applies the style to the input fields.

Use <form data-validate> for validation of the form. It uses standard html validation attributes such as pattern type min max etc.

Remember to also include the validation script for this functionality

Informative description (a <small> gets this styling by default)
<div class="form-group"> <label for="Name">Required textfield</label> <input id="Name" type="text" required> <small>Informative description</small> </div>

Form sizing

By default the input is 100% width, so use the grid for sizing the inputs.

<div class="row"> <div class="col-md-3"> <div class="form-group"> ... </div> </div> <div class="col-md-9"> <div class="form-group"> ... </div> </div> </div>
